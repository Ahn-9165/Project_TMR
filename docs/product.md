# Product Notes

## MVP Features

- Current weather lookup by location
- Rule-based anju and drink recommendation
- Nearby restaurant search by keyword
- Recommendation detail page
- Favorites and recent recommendations later

## Core Screens

- Home: weather summary, recommendation card, nearby restaurants
- Recommendation Detail: reason, matching drink, tags, spice level, calories later
- Nearby Restaurants: map and list view
- Favorites: saved menus and recent picks

## Recommendation Examples

| Condition | Food | Drink | Reason |
|---|---|---|---|
| Rain | Seafood pajeon | Makgeolli | Rainy weather pairing |
| Cold | Odeng-tang | Soju | Warm soup for cold weather |
| Hot | Sashimi | Beer or soju | Light cold food for hot weather |
| Any | Fried chicken | Beer | Reliable default option |
